# Portfolio-Mithsd
Portfólio responsivo feito com tecnologias web nativas (Vanilla JS/CSS) para Mithsd.

👉 [**Clique aqui para ver o projeto em produção (Vercel)**](https://portfolio-mithsd.vercel.app)

---

## Tecnologias Utilizadas

* **HTML5** (Estruturação semântica e acessível)
* **CSS3** (Variáveis nativas, CSS Grid, CSS Masonry, Clip-Path e curvas Cubic-Bezier)
* **JavaScript Vanilla** (Manipulação de DOM, APIs de eventos de mouse e IntersectionObserver)

---

## Estrutura de Arquivos

```text
├── index.html       # Estrutura semântica unificada
├── style.css        # Variáveis de cores, layouts responsivos e animações
├── script.js        # Lógica do menu, drag-and-drop e IntersectionObserver
└── src/             # Ativos locais (GIFs, backgrounds e imagens)
